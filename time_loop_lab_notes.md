# Time Loop Lab Notes

This is now a partial draft centered on `Rintaro Okabe` as the protagonist, with the main colleagues rewritten as Steins;Gate characters inside a temporal research facility.

## Current Story Direction

- Protagonist: `Rintaro Okabe`
- Key colleagues in the current draft:
  - `Kurisu Makise`
  - `Itaru "Daru" Hashida`
  - `Mayuri Shiina`
  - `Professor Leskinen` as the authority figure overseeing the experiment
- Premise:
  - Halcyon Temporal Research Center is running a dangerous resonance trial.
  - Okabe experiences the Core event, wakes back at `05:43`, and realizes the day is repeating.
  - The current draft ends once Okabe, Kurisu, and Daru uncover the first crucial clue:
    - `ANCHOR CONTINUITY // WITNESS RETENTION`
    - the hidden access route: `LOWER SHAFT / ANCHOR NODE / ACCESS TREE SIGMA-3`

## Scope Of This Draft

This version is intentionally a mid-draft rather than a full submission.

It currently covers:
- setup at the lab
- introduction of the Steins;Gate cast in this setting
- the first collapse/reset
- Okabe's confirmation that he is in a loop
- the first major breakthrough toward breaking it

It does not yet cover:
- the deeper loop investigation
- multiple route branches
- full ending paths
- the final confrontation and loop resolution
